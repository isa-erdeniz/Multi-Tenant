Garment Core ile güçlendirildi

Abonelik planları

Ödeme formu iyzico güvenli altyapısındadır. Yetkilendirme ve plan hakları garment_core API üzerinden yönetilir.

Fiyatlar /api/pricing (D1) üzerinden yüklendi.

Yükleniyor…

1. Garment Core hesabı

JWT almak için e-posta ve şifrenizi girin (DressifyeUser).

2. Ünvan (isteğe bağlı)

Profilinizde görünecek özel başlık (garment_core UserProfile).

3. Ödeme formu

Önce yukarıdan plan seçin, giriş yapın; ardından form oluşturulur.

Callback: dressifye.com/odeme/abonelik-callback/ → Worker → garment_core doğrulama. Webhook imzaları yalnızca backend’de (X-Iyz-Signature-V3) doğrulanır.
